Sobre

Esse site(página DIO) consiste em 3 diferentes páginas, sendo eles:

  • Página Home
  • Página Cadastro
  • Página Login
  • Página de Feed

A página de Home apenas possui botões que consegue levar o usuário para as outras páginas

A página de Cadastro recebe uma validação de nome de usuário, e-mail e senha, que quando não aceitas devolvem uma mensagem de erro

OBS: A página não aceita novos usuário

A página de Login simula uma API usando JSON-server, caso, as informações corretas são passadas se consegue entrar na página de Feed do usuário
